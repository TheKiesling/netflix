## Usar el proyecto
1. Clonar el repositorio
```
git clone git@github.com:TheKiesling/netflix.git
```
2. Dirigirse al directorio del proyecto
```
cd netflix
```
3. Instalar yarn
```
yarn
```
4. Instalar las dependencias 
```
yarn install
```

## Scripts
- Ejecutar el proyecto 
```
yarn dev
```
- Evaluar Linting
```
yarn lint
```
